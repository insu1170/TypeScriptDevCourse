# 21일차

## 데이터베이스

데이터를 통합하여 효율적으로 관리하기 위한 데이터 집합체

- 데이터 구조화 및 관리
- 데이터 중복 최소화 및 효율적인 데이터 관리

## DBMS

데이터베이스 운영 관리를 위한 시스템

## SQL

데이터베이스 연산 요청을 위해 사용되는 언어로 데이터 생성·조회·수정·삭제 같은 작업 가능

- 삽입: INSERT
- 조회: SELECT
- 수정: UPDATE
- 삭제: DELETE

## RDBMS

관계형 데이터베이스로, 테이블 간 관계 설정을 통해 필요한 데이터를 찾을 수 있다.

- 기본키 (PK)
    - 해당 테이블의 각 행을 유니크하게 구별할 수 있는 키 값
- 외래키 (FK)
    - 다른 테이블의 기본키를 참조하여 데이터를 연결하는 키 값

### 연관관계

- 1:1 관계
- 1:N 관계
- M:N 관계
