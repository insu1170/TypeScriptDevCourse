# 인터넷과 웹 기초 개념

## 인터넷(Internet)

- 전 세계 컴퓨터를 하나의 통신망으로 연결한 네트워크
- `International Network`의 약자

---

## 웹(Web)

- 인터넷에 연결된 컴퓨터를 통해 사람들이 정보를 공유하는 공간
- **WWW (World Wide Web)**
- 관계: **인터넷 ⊃ 웹**
- 팀 버너스-리가 연구원 간 정보 교환을 위해 고안
- 정보를 **하이퍼텍스트** 형식으로 표현하여 문서 간 이동 가능
- 웹은 **HTML + CSS + JavaScript**로 구성됨

---

## 클라이언트 & 서버

- **클라이언트(Client)**
    - 서비스를 이용하는 컴퓨터 또는 프로그램
- **서버(Server)**
    - 서비스를 제공하는 컴퓨터 또는 프로그램
- **프로토콜(Protocol)**
    - 서버와 클라이언트가 통신할 때 지켜야 하는 규칙
    - 웹에서는 **HTTP 프로토콜**을 통해 데이터 통신

---

## 프론트엔드(Frontend)

- 웹 서비스에서 사용자에게 보이는 영역
- 그래픽 사용자 인터페이스(GUI)를 담당
- 사용자와의 직접적인 상호작용 처리

---

## 백엔드(Backend)

- 웹 서비스에서 사용자에게 보이지 않는 서버 측 영역
- 프론트엔드에서 전달받은 요청과 데이터를 처리
- 데이터베이스 및 서버 로직과 연산 수행

---

- 개발에 필요한 도구를 하나로 제공하는 환경

### Visual Studio Code

- 높은 호환성과 확장성으로 가장 많이 사용됨

---

## HTML

- 하이퍼텍스트를 기반으로 웹 페이지를 구성하는 언어
- 웹 페이지의 **구조를 정의**하는 마크업 언어(Markup Language)

### 특징

- 태그(`< >`)를 사용하여 구성
- 대부분 **여는 태그와 닫는 태그**로 이루어짐

### HTML 기본 구조

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
</head>
<body>

</body>
</html>

```
