# CSS & JavaScript 기초 개념

## CSS

- HTML을 꾸며주는 스타일 언어
- 문서 전체를 한 번에 꾸미는 것이 아니라 **각 태그(요소) 단위로 스타일 적용**

### CSS 적용 방식 3가지

1. **인라인 스타일**
    - HTML 태그 내부에 직접 작성
2. **내부 스타일**
    - HTML 문서의 `<style>` 태그 안에 작성
3. **외부 스타일**
    - CSS 파일을 별도로 작성 후 HTML 문서에 연결

---

## JavaScript

- 웹 페이지에서 **특정 요소를 선택하여 제어**할 수 있는 스크립트 언어
- 사용자와의 상호작용 및 동적인 기능을 담당
- 최근에는 **백엔드 언어(Node.js)** 로도 널리 사용됨

### JavaScript 작성 방식

- CSS와 마찬가지로
    - 인라인 스크립트
    - 내부 스크립트
    - 외부 스크립트
        
        방식으로 작성 가능
        

---

## 스크립트 언어란?

- 독립적으로 실행되는 언어라기보다는
    
    **기존 프로그램을 제어하는 역할**을 하는 언어
    
- 최근에는 빠른 런타임 환경의 발전으로
    
    스크립트 언어만으로도 충분한 프로그래밍이 가능해짐

## JavaScript 기본 문법 개념

### 함수(Function)

- 특정 기능을 수행하는 코드 묶음

### 조건문 (if 문)

- 조건에 따라 다른 동작을 수행하도록 분기 처리하는 문법

### 변수(Variable)

- 값을 저장하는 공간

### JavaScript 변수 선언 키워드

- `const`
    - 상수
    - 한 번 선언하면 값 변경 불가
- `let`
    - 값 변경 가능
- `var`
    - 함수 스코프 문제로 인해 **사용을 지양**
