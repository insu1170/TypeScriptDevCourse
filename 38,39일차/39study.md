## 모듈화 필요성

코드의 퀄리티를 높이고 유지보수를 쉽게 하기 위해 모듈화를 진행한다.

## 패키지 구조

### Router

HTTP Method와 요청 경로에 따라 어떤 로직이 실행될지 연결해준다.

### Controller

요청을 받아 Service로 전달하고 결과를 응답으로 반환한다.

### Service

실제 비즈니스 로직을 처리하고 어떤 쿼리를 실행할지 결정한다.

### Model

DB와 직접 소통하는 계층이며 SQL 쿼리 집합이 위치한다.

## 추가적으로 필요한 것

### 예외 처리

에러 상황을 관리하고 적절한 응답을 반환하도록 한다.

### 유효성 검사

요청 데이터가 올바른지 검증한다.

### JWT

- Access Token : 로그인 인증에 사용
- Refresh Token : Access Token 만료 시 재발급에 사용

### Faker

임의의 데이터를 생성하여 테스트용 더미 데이터를 만들 때 사용한다.
