# 19일차

## 빈 객체 확인 방법

### 1. Object.keys() 사용

```jsx
Object.keys(obj).length === 0

```

- 가장 흔히 사용하는 방법
- 열거 가능한 key 기준

---

### 2. for...in 으로 확인

```jsx
let isEmpty = true;
for (let key in obj) {
  isEmpty = false;
  break;
}

```

- prototype 체인까지 순회 가능 (주의)

---

### 3. lodash 라이브러리 사용

```jsx
_.isEmpty(obj)

```

- 객체, 배열, Map, Set 모두 가능
- 외부 라이브러리 의존성 발생

---

## Map 객체 확인

```jsx
map.size === 0

```

- Map / Set 공통
